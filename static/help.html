DeepAgentForce 使用手册

欢迎使用 DeepAgentForce 智能助手平台！本手册涵盖从快速入门到高级功能的完整指南。使用左侧目录或 Ctrl+K 搜索快速定位内容。

🚀 快速开始

1

注册并登录

首次访问系统，点击注册创建账号，或使用已有账号登录。登录后将进入主界面。

2

配置大模型

点击左侧「配置」菜单，填写模型 API 地址、API Key 和模型名称。支持 OpenAI、OpenRouter 等兼容接口。

3

开始对话

在主对话界面输入问题，AI 将自动分析意图并生成回答。你还可以上传文件、构建知识库、使用各种工具技能。

提示：系统支持多会话管理，每次对话会自动保存。你可以通过左侧栏查看历史会话，随时继续之前的讨论。

💬 智能对话

图 2：智能对话界面

基础 如何提问

在底部的输入框中输入你的问题，按 Enter 或点击发送按钮即可。AI 会自动分析问题并生成回答。

进阶 上传文件

点击输入框左侧的 📎 图标，可上传 PDF、Word、TXT、Markdown 等格式文件。AI 会读取文件内容并结合回答。

技巧 多会话管理

点击左侧的「+」按钮可开启新对话。所有对话会自动保存在历史记录中，方便随时回顾和继续。

📚 知识库管理

构建专属知识库

进入「知识库」页面，上传你的私有文档（PDF、Word、TXT、Markdown），系统会自动向量化并建立索引。之后的相关问题会优先检索你的知识库内容。

格式 说明
PDF 最常用的文档格式，支持文本和图片提取
Word (.docx) 微软 Office 文档，支持段落和表格
TXT 纯文本文件，最简单可靠
Markdown 支持代码和格式的轻量级标记语言

图 3：知识库管理界面

注意：上传文档后，系统需要一定时间处理和索引。处理完成后即可在对话中检索使用。

📥 文件下载

下载 AI 生成的文件

当 AI 生成文件（如作文、报告、代码等）时，可以要求 AI 保存到指定位置。文件会自动保存到你的下载目录中。

1

请求 AI 保存文件

例如：「帮我写一篇关于春天的作文，并保存到文件」

2

AI 自动保存

AI 会将文件保存到 data/outputs/{租户ID}/ 目录

3

点击下载

点击页面右上角的「下载」按钮，在弹出的列表中查看和下载文件

提示：每个租户的文件是隔离存储的，只能查看和下载自己账号生成的文件。

⚙️ 模型配置

配置说明

在「配置」页面填写大模型的连接信息。配置会自动保存，下次启动时无需重新填写。

配置项 说明 示例
LLM URL 模型 API 的 Base 地址 https://openrouter.ai/api/v1
LLM API Key 访问模型的密钥 sk-or-xxxxx
LLM Model 使用的模型名称 anthropic/claude-3.5-sonnet
提示：URL 只需填写 Base 地址（如 https://openrouter.ai/api/v1），系统会自动拼接正确的接口路径。

❓ 常见问题

提示"未连接到服务器"怎么办？
确保后端服务正在运行（执行 python main.py）。检查网络连接是否正常。
AI 回复很慢或无响应？
可能是模型 API 响应较慢，或网络连接不稳定。可以检查「配置」页面的模型设置是否正确。
如何下载 AI 生成的文件？
点击页面右上角的「下载」按钮。确保在对话中让 AI 保存了文件（如「帮我保存这篇作文」）。
多租户是什么意思？
每个账号（租户）有独立的数据空间，包括上传的文档、生成的配置文件等。不同租户之间数据完全隔离，确保隐私安全。
如何让 AI 使用我的文档回答？
在「知识库」页面先上传文档，之后的对话会自动检索相关内容。
知识库搜索不准确？
确保文档内容清晰、格式规范。复杂 PDF 建议先转换为纯文本格式上传。
